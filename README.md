
# Process management-automation tool.

## A simple project written in Java to create templates with custom type variables data.

This project is a baseline which allows you to create Data templates with custom type variables. From this templates you can create "Prospects" with data validation. Every part of this project can be used as sample code which shows how to do the following:

* How to create custom R2DBC JSON Converters.
* How to write end-to-end tests using R2DBC webTestController, with in-memory H2 database.
* How to setup Testcontainers with R2DBC and Liquibase, which allow tests to run with real database docker instances (eg. PostgreSQL).
